How to install

In this file, bookmarklets are offered as links (with the javascript: method prefix). To install them in your browser, simply drag-and-drop the link to your bookmarks bar.

If you install more than a few of them, it is probably a good idea to create a folder to organize them in a more compact way.

Text insertion

The following scripts insert text into a text input field. Obviously, they only work if such an input field is active. They may not work in web applications that use other means of text input (e.g. the Twitter web client uses a div as input field.

This bookmarklet inserts the current date and/or time at the cursor position in any text input field.
