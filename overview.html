Sascha’s Bookmarklets

Introduction

This is a collection of Bookmarklet scripts that I created for my own use. They are published here for others to use and enjoy.

Please note that while I try to make them as cross-platform and standard-compliant as possible, I am in fact mostly using Firefox and I rarely test anything on any other browser. If I know about any incompatibilites, I list them below, but it is likely that these lists are not exhaustive.

Regardless of my own testing, all the scripts are provided “as is”, i.e. no guarantees for proper functioning or otherwise are given, and of course I take no responsibility if things go wrong for whatever reason.

How to install

In this file, bookmarklets are offered as links (with the javascript: method prefix). To install them in your browser, simply drag-and-drop the link to your bookmarks bar.

If you install more than a few of them, it is probably a good idea to create a folder to organize them in a more compact way.

Text insertion

The following scripts insert text into a text input field. Obviously, they only work if such an input field is active. They may not work in web applications that use other means of text input (e.g. the Twitter web client uses a div as input field.

Insert current Date/Time (formatted)

This bookmarklet inserts the current date and/or time at the cursor position in any text input field.

Unicode Text

These bookmarklets make use of the fact that most websites nowadays use UTF-8 or another Unicode character encoding. That means that you have many interesting and useful – but often hard to reach – characters available for your writing.

Character replacements

This is a set of bookmarklets that replace typed (Latin) text with characters that represent the same letters but have a different visual appearance:

Bookmarklets:
Convert plain text to Sᴍᴀʟʟ Cᴀᴩs
Convert plain text to Ｆｕｌｌｗｉｄｔｈ
Convert plain text to Ⓒⓘⓡⓒⓛⓔⓓ
Convert plain text to ⒫⒜⒭⒠⒩⒯⒣⒠⒮⒤⒮⒠⒟
Convert plain text to ˢᵘᵖᵉʳˢᶜʳⁱᵖᵗ

Math Pseudofont substitutions

The Unicode standard defines a number of styled character sets for mathematical typesetting. The main difference to the other character replacements is that these are using characters from the SMP (Supplementary Multilingual Plane), which isn't supported on all platforms (notably, Android and iOS-users won't see them).
