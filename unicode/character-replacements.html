Unicode Character Replacements

This is a set of bookmarklets that replace typed (Latin) text with characters that represent the same letters but have a different visual appearance.

Detailled Information

Small Caps

There are several symbols available in Unicode that resemble small capitals. This script replaces all lower-case plain characters (i.e. without accents) with these characters to create the appearance of a proper Sᴍᴀʟʟ Cᴀᴩs text.

Depending on the typeface the visual appearance of these characters may slightly vary, but they appear to work well on all common platforms.

Example:
Lᴏʀᴇᴍ ɪᴩsᴜᴍ ᴅᴏʟᴏʀ sɪᴛ ᴀᴍᴇᴛ, ᴄᴏɴsᴇᴛᴇᴛᴜʀ …

Superscript

In addition to a full set (0-9) of numerals, Unicode specifies an almost complete set of superscript characters, including upper- and lower-case letters. Only the letter Q is missing from both sets.

Since these characters come from different contexts they may vary in appearance, but they appear to work just fine on all common platforms.

Example:
ᴸᵒʳᵉᵐ ⁱᵖˢᵘᵐ ᵈᵒˡᵒʳ ˢⁱᵗ ᵃᵐᵉᵗ, ᵐᵃᵍⁿᵃ ᵃˡⁱqᵘʸᵃᵐ ᵉʳᵃᵗ…

Circled

The Unicode “Enclosed Alphanumerics” code block contains a full set of encircled letters and numbers. For good measure, the script also replaces a number of mathematical operators and the katakana Tu (ツ) with encircled versions

Support for “Enclosed Alphanumerics” is very good across browsers and platforms, but the encircled math operators may look slightly different from the other characters, depending on the platform.

Example:
Ⓛⓞⓡⓔⓜ ⓘⓟⓢⓤⓜ ⓓⓞⓛⓞⓡ ⓢⓘⓣ ⓐⓜⓔⓣ, ㋡

Parenthesised

Also part of the “Enclosed Alphanumerics” code block is a set of letters and all numbers with parentheses. There are no upper-case variants of these, therefore the script also converts all letters to lower-case.

Support for “Enclosed Alphanumerics” is very good across browsers and platforms.

Example:
⒧⒪⒭⒠⒨ ⒤⒫⒮⒰⒨ ⒟⒪⒧⒪⒭ ⒮⒤⒯ ⒜⒨⒠⒯ …

Fullwidth Forms

These are variants of the standard plain Latin characters

Support for Fullwidth Forms is very good across browsers and platforms. Google even seems to recognise them as equivalent to the regular characters, at least for searches.

Example:
Ｌｏｒｅｍ　ｉｐｓｕｍ　ｄｏｌｏｒ　ｓｉｔ　ａｍｅｔ　…
